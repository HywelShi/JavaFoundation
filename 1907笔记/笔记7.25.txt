案例2：
问：536-5245之间有没有被1689整除的数，有的话输出来
int a = 1;
for(int i=536;i<=5245;i++){
	if(i%1689==0){
	  a=2;
	  System.out.println(i);
	}
}
if(a==1){
	System.out.println("没有！");
}else{
	System.out.println("有！");
}
